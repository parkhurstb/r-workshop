nake my own first graph

Were going to a make a line plot of year (x-variable) vs visitors (y-variable) in ggplot:

ggplot(data = ci_np, aes(x = year, y = visitors)) +
  geom_line()

#can use geom_point or goem_area for different types of graphs

Make a base layer for my CI plots:

gg_base <- ggplot(data = ci_np, aes(x = year, y = visitors)) +
  geom_line()

gg_base + geom_point()

# Now lets start customizing! - line / point colors: color - change line type: linetype -change point style: pch - Change transparency: alpha

gg_base + 
  geom_line(
    color = "purple"
  )
